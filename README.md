# 💰 Simple Interest Calculator (Bash)

This project is a simple Bash script that calculates simple interest based on user input. It demonstrates basic shell scripting, user input handling, and arithmetic operations in a Linux environment. The script takes the principal amount, interest rate, and time period from the user and computes the result using the formula: Simple Interest = Principal × Rate × Time.

## 📌 Features
- Accepts user input for principal, rate, and time  
- Performs calculation using a Bash script  
- Displays the result in the terminal  

## 🛠️ Technologies Used
Bash (Shell Scripting), Linux Terminal, and the `bc` command-line calculator.

## 🚀 How to Run
Clone the repository using:  
git clone (https://github.com/Princess2003/Git_project)  
Then navigate into the folder:  
cd Git_project  
Make the script executable:  
chmod +x simple_interest.sh  
Run the script:  
./simple_interest.sh  

## 🧮 Example
Enter the principal amount: 1000  
Enter the annual interest rate (as a decimal): 0.05  
Enter the time period (in years): 2  
Simple Interest: 100
